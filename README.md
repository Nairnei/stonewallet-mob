# Stone Wallet

A small project that allows trading in Brita, Bitcoin and Real
<br>
- Brita can swap to Bitcoin;
- Bitcoin can swap to Brita;
- Real buy Brita or Bitcoin;
- Brita or Bitcoin can be sold in Real.
